# Instagram Clone (MERN Project)

A full-stack Instagram clone built using MongoDB, Express.js, React, and Node.js. 
Allows users to create accounts, post images, view stories, follow/unfollow profiles, and manage their own profile.

## Features
- User registration and login
- Create, view, and delete posts
- Stories viewing
- Profile management
- Follow/Unfollow users
- Suggestions for new users

## Tech Stack
- Frontend: React, Bootstrap
- Backend: Node.js, Express.js
- Database: JSON file (or MongoDB if upgraded)
- Version Control: Git
- Hosting: GitHub

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ManojKumarJ1902/Instagram-Clone-Mern-Project.git
2. Navigate to project folder:
   ````bash
   cd Instagram-Clone-Mern-Project
3. Install dependencies:
   ````bash
   npm install
4. Start the server:
   ````bash
   npm start
5. Open in browser: http://localhost:3000
